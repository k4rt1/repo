Особенности названий переменных:
	1) 'n' перед названием говорит о количестве обьекта( nStudents )
	2) 'No' после названия говорит о номере обьекта и ( studentNo )
	3) '_' после названия у переменной класса показывает, что она находится в зоне области private ( size_ )
